## 📌 About

InterviewIQ AI is a full-stack AI interview preparation platform built to help users prepare for real technical interviews.

The goal is to create a modern platform where users can upload their resume, practice AI-generated interviews, receive detailed feedback, and track their improvement over time.

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/Hiba-khan01/InterviewIQ-AI.git
cd InterviewIQ-AI
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend

python -m venv .venv

# Windows
.venv\Scripts\activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```

Frontend runs on:

```
http://localhost:3000
```

Backend runs on:

```
http://127.0.0.1:8000
```

Swagger Docs:

```
http://127.0.0.1:8000/docs
```
